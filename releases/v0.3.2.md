# v0.3.2

> Released: 2026-03-19

## Highlights

- **CLI command transition hardening** - `cubecloud.io` is now validated as the primary command while preserving `paperclipai` as a compatibility alias.
- **Environment variable migration safety** - `CUBECLOUDIO_*` keys remain preferred, with `PAPERCLIP_*` fallback behavior preserved and covered by tests.

## Improvements

- **Alias regression coverage (CLI)** - Added tests to verify CLI bin metadata contains both `cubecloud.io` and `paperclipai`, and the runtime command name remains `cubecloud.io`.
- **Alias regression coverage (env vars)** - Expanded env alias tests to assert warning-on-fallback semantics, including one-time warnings per legacy key.
- **Adapter utilities parity** - Added env alias behavior tests in `@cubeclawio/adapter-utils` so fallback/warning behavior is validated outside CLI/server packages.

## Verification

Validated after latest changes with:

- `corepack pnpm -r typecheck`
- `corepack pnpm test:run`
- `corepack pnpm build`

Result: all commands passed.
